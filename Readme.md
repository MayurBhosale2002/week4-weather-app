# Week 4 Weather Application

Name: Mayur Bhosale  

## Description
A responsive weather application that displays real-time
weather data using OpenWeather API.

## Features
- City search
- 5-day forecast
- Celsius/Fahrenheit toggle
- Local storage
- Responsive design

## Technologies
- HTML
- CSS
- JavaScript
- REST API

## Setup Instructions
1. Get API key from OpenWeather
2. Paste it in js/config.js
3. Open index.html in browser
4. Search any city

## Folder Structure
Follows standard modular structure.
